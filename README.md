# Roda Repasse

Sistema de cálculo e gestão de repasse para PDVs — Roda Conveniência.

## Stack
- React 18 + Vite
- Supabase (PostgreSQL + Auth)
- GitHub Pages (hospedagem)

## Desenvolvimento local

```bash
npm install
npm run dev
```

Abre em `http://localhost:5173`

## Deploy

Push para `main` dispara o workflow `.github/workflows/deploy.yml` que builda e publica em GitHub Pages.

## Acesso

Solicite credenciais ao administrador do sistema.
